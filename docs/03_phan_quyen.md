# Phan quyen

## Vai tro de xuat
- Admin: toan quyen, quan ly danh muc, nhan su, phan quyen, cau hinh.
- Manager: xem/duyet bao cao, chinh sua gia phong, xem thong ke.
- Receptionist: dat phong, check-in/out, quan ly khach hang.
- Accountant: hoa don, thanh toan, bao cao doanh thu.

## Nhom quyen theo chuc nang
- RoomManagement: Quan ly phong/loai phong.
- BookingManagement: Dat phong, gan phong, check-in/out.
- CustomerManagement: Quan ly khach hang.
- ServiceManagement: Quan ly dich vu.
- InvoicePayment: Lap hoa don, thanh toan.
- Reporting: Xem bao cao doanh thu, cong suat.
- UserManagement: Nhan su, tai khoan, phan quyen.
